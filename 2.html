CSS

선택자
  • style=""안에 쓰면 css로 효과가 적용된다! 딱 하나 개별 적용할 때 쓰기 굿
  • 밑줄 없애고 싶으면 text-decoration: none하는 것이고 밑줄 넣고 싶으면 : underline하면 된다
  • css 코드 찾아볼 때는 'css ~~ property' 일케 찾아보면 된다->보통 W3C 홈피에서 잘 알려줌
  • class="#"해서 class 설정하고 css에서 이거 지정할 때는 .#으로 해준다. class는 앞에 온점!
  • 하나에 class 여러 개를 설정할 수 있으며, 띄어쓰기로 구분한다. ex. class="saw active"는 saw css도 적용되고 active css도 적용된다. 코드에서 더 뒤에 있는 코드로 적용
  • class와 비슷한 얘가 id다. id는 앞에 #을 붙인다. ex. id="just" #just // id는 class와 다르게 순서와 상관 없이 만약 class와 중복 적용된 상황이라면 코드 순서가 어떻든 무조건 id로 css가 적용된다.
  • 선택자: tag(앞에 아무것도 없이 입력하는 것)가 가장 약하고 다음이 class, 가장 센 얘가 id다.
  • class는 중복해서 써도 좋지만 id는 딱 하나만 써야한다. ex. class="saw"를 여러 번 써도 되지만 id="active"를 여러 번 쓰면 안 된다. id="different"와 같은 다른 id 값은 당연히 써도 된다.
